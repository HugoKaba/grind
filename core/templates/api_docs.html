Use JWT Token in Authorization header:

Get Token:

POST /api/auth/token/

📝 Tweets

GET /tweets/
List all tweets

Get paginated list of all tweets (20 per page)

Response 200:
{
    "count": 45,
    "next": "http://localhost:8000/api/tweets/?page=2",
    "results": [
        {
            "id": 1,
            "author": {"id": 1, "username": "messi"},
            "content": "Goal!",
            "likes_count": 150,
            "retweets_count": 45,
            "replies_count": 12,
            "created_at": "2026-06-16T12:00:00Z"
        }
    ]
}
POST /tweets/
Create a new tweet

Requires authentication

Request Body:
{
    "content": "Amazing goal! #football"
}
Response 201:
{
    "id": 46,
    "author": {"id": 1, "username": "messi"},
    "content": "Amazing goal! #football",
    "likes_count": 0,
    "retweets_count": 0,
    "replies_count": 0,
    "created_at": "2026-06-18T09:00:00Z"
}
GET /tweets/{id}/
Get tweet by ID
Example: /tweets/1/
{
    "id": 1,
    "author": {"id": 1, "username": "messi", "profile": {...}},
    "content": "Goal!",
    "likes_count": 150,
    "liked_by": [2, 3, 4],
    "retweeted_by": [5, 6],
    "replies": [
        {"id": 2, "author": {...}, "content": "Amazing!"}
    ]
}
POST /tweets/{id}/like/
Like a tweet

Requires authentication

Example: /tweets/1/like/
Response 200:
{
    "likes_count": 151,
    "liked": true
}
POST /tweets/{id}/retweet/
Retweet a post

Requires authentication

💬 Messages

GET /messages/
List all conversations

Requires authentication - Shows only followed users

{
    "count": 5,
    "results": [
        {
            "id": 1,
            "sender": {"id": 1, "username": "messi"},
            "recipient": {"id": 2, "username": "ronaldo"},
            "content": "Hey!",
            "is_read": true,
            "created_at": "2026-06-18T09:00:00Z"
        }
    ]
}
POST /messages/
Send a message

Requires authentication

Request Body:
{
    "recipient_id": 2,
    "content": "Hey, how are you?"
}
GET /messages/{username}/
Get conversation with user

Requires authentication - Can message any user

⚠️ Error Codes

400 - Bad Request

Invalid request data or parameters

401 - Unauthorized

Missing or invalid authentication token

403 - Forbidden

Authenticated but not authorized to perform action

404 - Not Found

Resource not found

500 - Server Error

Internal server error
